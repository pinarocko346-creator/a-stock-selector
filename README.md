# A股选股系统 - 基于13个买点策略

一个基于道氏理论、威科夫理论和波浪理论的A股技术分析选股系统。

## 📊 13个买点策略

### 底部反转结构的买点（1-4）

1. **2B结构（云聪版）**
   - 完整的五浪下跌
   - 大阴线跌破平台后大阳线快速拉起
   - 概率高达90%

2. **底部头肩的右肩（50%位置）**
   - 价格回调到前一个下跌波段的50%位置
   - 提前埋伏买入

3. **二次测试**
   - 刺破前低但很快收回
   - 伴有明确K线信号（长下影、反包）

4. **强势出现（威科夫概念）**
   - 底部形态中轴以上
   - 出现带量阳K线

### 形态突破和中继的买点（5-8）

5. **趋势线 + 量能密集区的突破**
6. **趋势线 + 关键点位的突破**
7. **趋势线 + 形态的突破**
8. **大形态突破**
   - 长期底部盘整后的突破
   - 茶杯带柄、老鸭头模型

### 上涨趋势中的买点（9-13）

9. **波段的回踩（斐波那契回调）**
   - 回调到0.382、0.5或0.618位置

10. **箱体/形态内的弹簧**
    - 跌破箱体底部但快速拉回

11. **多周期共振（趋势加速）**
    - 多个周期都呈现上涨趋势

12. **支撑压力互换**
    - 前期压力转换为支撑

13. **趋势急跌（首次急跌到50%）**
    - 上涨趋势中第一次快速暴跌

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install pandas numpy akshare
```

### 2. 运行选股程序

```bash
cd ~/a-stock-selector
python main.py
```

### 3. 选择模式

- **快速测试模式**：分析10只示例股票（推荐先用这个测试）
- **完整模式**：分析所有A股（需要较长时间）

### 4. 查看结果

程序会自动生成CSV文件，包含：
- 股票代码
- 股票名称
- 符合买点数量
- 选股原因（详细列出所有符合的买点）
- 最新价格
- 分析日期

你可以修改 `main.py` 中的股票代码列表，只分析特定的股票：

```python
custom_codes = ['000001', '600519', '000333']
selector.select_stocks(stock_codes=custom_codes)
```

## 📚 理论基础

本系统基于以下经典技术分析理论：

1. **道氏理论**：趋势、支撑压力、形态
2. **威科夫理论**：量价结构、吸筹派发
3. **波浪理论**：斐波那契回调、波浪结构
